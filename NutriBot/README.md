# NutriBot

Aplicativo Android de nutrição com chatbot integrado usando Jetpack Compose, Room, Retrofit e Hilt.

## Estrutura do Projeto

- **data/**: Camada de dados (Room, API, Repositórios)
- **domain/**: Modelos de domínio e casos de uso
- **presentation/**: UI com Jetpack Compose
- **di/**: Injeção de dependência com Hilt

## Tecnologias

- Kotlin
- Jetpack Compose
- Room Database
- Retrofit
- Hilt (DI)
- Firebase Auth
- MPAndroidChart
